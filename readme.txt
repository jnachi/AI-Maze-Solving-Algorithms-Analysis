
To execute the BFS_DFS_Astar.py and mdp.py scripts with 35 as an argument, follow these steps:

1. Ensure you have Python installed on your system.
2. Install the required libraries by running the command:
   pip install -r requirements.txt

To run BFS_DFS_Astar.py:
-------------------------
Open a terminal or command prompt and navigate to the directory containing the BFS_DFS_Astar.py file. You can change the argument which is the maze size for visualisation. Then, run the following command:

python BFS_DFS_Astar.py 35

To run mdp.py:
--------------
Open a terminal or command prompt and navigate to the directory containing the mdp.py file.
You can change the argument which is the maze size for visualisation. Then, run the following command:

python mdp.py 35

These commands will execute the respective Python scripts with 35 passed as an argument to each.
